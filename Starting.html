Početak rada

Kako dodati React na vašu web stranicu? Veoma jednostavno.

Isprobajte React

React je napravljen sa postepenom adaptacijom u vidu, tako da možete koristiti baš onoliko React-a koliko vam treba za vašu aplikaciju. Bilo da samo želite da isprobate React, dodate nekoliko interaktivnih elemenata na vašu web stranicu ili kreirate kompleksnu React aplikaciju, linkovi u ovoj sekciji će vam pomoći da počnete.

Dodajte React Na Vašu Web Stranicu

Možete dodati React na vašu stranicu za manje od 60 sekundi. Potom ga možete postepeno proširivati ili jednostavno dodati par jednostavnih interaktivnih komponenti koje će ulepšati vašu web stranicu.

Korak 1: Dodajte DOM Container u vaš HTML

Prvo, otvorite HTML dokument koji u koji želite da dodate React. Dodajte prazan <div> tag na mesto gde želite da prikažete nešto uz pomoć React-a. Na primer:

<!--Vaš HTML-->

<div id="like_button_container></div>

<!--Vaš HTML-->

Dali smo ovom <div> tagu jedinstven id HTML atribut. Ovo će nam kasnije omogućiti da ga pronađemo iz Javascript koda i smestimo React sadržaj unutar njega.

Korak 2: Dodajte Script Tagove

Sledeći korak, dodajte tri <script> taga u vaš HTML pre taga za zatvaranje </body>:

<script src="https://unpkg.com/react@16/umd/react.development.js" crossorigin><script>

<script src="https://unpkg.com/react@16/umd/react-dom.development.js" crossorigin><script>

<script src="like_button.js"><script>

Prva dva taga će učitati react. Treći tag je vaš Javascript kod.

Step 3: Kreirajte React Komponentu

Kreirajte fajč like_button.js u folderu vaše HTML stranice.

Otvorite ovaj Primer Koda i prekopirajte ga u fajl koji ste upravo kreirali.

Ispod koda koji je dat u primeru, dodajte dve linije koda u like_button.js:

// ... Kod koji ste kopirali ...

const = document.querySelector('#like_button_container');

ReactDOM.render(e(LikeButton), domContainer);

Ove dve linije koda pronalaze <div> tag koji smo dodali u naš HTML u prvom koraku, a potom prikazuje našu “Like” React komponentu unutar nejga.

Korak 4: Gotovo!

Nema četvrtog koraka. Upravo se uspešno dodali vašu prvu React komponentu na web stranicu.
