React Ruter

Single Page Applications

React.js je JavaScript biblioteka otvorenog koda koja se koristi za izradu single-page aplikacija. Koristi se za kreiranje interfejsa za web i mobilne aplikacije.

React.js takođe omogućava programerima da kreiraju komponente korisničkog interfejsa za dugoročnu upotrebu. Razvijen je od strane Jordan Walke-a, softverski inženjer koji radi na Facebooku. Po prvi put je korišćen na društvenoj mreži Facebook 2011. i na Instagram 2012. godine. Trenutno se koristi kao alat u mnogim kompanijama, kao što su: Facebook, Instagram, Netflix, New York Times, Yahoo! i Dropbox. React ima živu zajednicu i veliku dokumentaciju, i iako je nešto teži za učenje, zaista se isplati naučiti ga. Kao deo React projekta, Facebook je razvio React Native koji radi u osnovi na isti način kao React, ali za izgradnju izvornih mobilnih aplikacija. React omogućava programerima da kreiraju velike web i mobilne aplikacije koje mogu da menjaju podatke, bez ponovnog učitavanja stranice. Glavna svrha React-a je da bude brz, skalabilan i jednostavan. Radi samo na korisničkim interfejsima u aplikaciji. Ovo odgovara prikazu u MVC šablonu. Može se koristiti u kombinaciji sa drugim JavaScript bibliotekama i okvirima, kao što je AngularJS.
