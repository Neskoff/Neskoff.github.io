React će vam omogućiti implementaciju novih funkcija bez izmene postojećeg korisničkog koda. React se takođe može implementirati i na severu koristeći Node i primeniti u mobilnim aplikacijama koristeći React Native.

Komponente u fokusu

Omogućava enkapsulaciju komponenata koje same prate svoje stanje i kombinuju se radi kreiranja kompleksih korisničkih interfejsa. Pošto je logika komponenata pisana u JavaScript-u a ne u šablonima(pattern-ima), omogućena je laka manipulacija DO modelom.

Deklarativan

React čini pravljenje kompleksnih korisničkih interfejsa jednostavnim. Dizajnirajte jednostavan pogled za svako stanje vaše aplikacije, a React će efikasno ažurirati i prikazati tacno određene komponente kada se vaši podaci promene.Deklarativni pogledi čine otklanjanje grešaka u vašem kodu mnogo jednostavnijim.
